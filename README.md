# Frame — 山を、作品に。

山の写真に山名・標高・解説を美しく重ねて、ポスターのような一枚に仕上げるウェブアプリ。
山を「記録として映す」のではなく、**一枚の作品として仕上げる**ことがテーマ。

約27,000座の山岳辞書から山を選ぶだけで、読み・英名・標高・解説が自動で添えられ、
テーマ（テンプレート）を選べば文字組み・余白・配置までまとまった仕上がりになる。

▶ https://takanarishimbo.github.io/frame/

[trace](../trace) の「山を写す(AR)」機能から派生したプロジェクト。3Dマップ・撮影地点・向き合わせ（AR投影）を取り除き、写真の仕上げに絞っている。

## 使い方（フロー）

写真一覧をハブにした自由な進行。順不同で再編集でき、最後にまとめて保存できる。

1. **写真を選ぶ** … ホームで手元の写真を選ぶ（複数可）。複数選ぶと1枚ずつ順に仕上げられる。
2. **山を選ぶ** … 山岳辞書（27,371座）から山名・読みで検索して、写真ごとに山を選ぶ（複数座も可）。
3. **テーマを選ぶ** … 雅 / 頂 / 語 / 間 / 空 の5種＋「素」（テーマなし）。スマホはスワイプ、PCはカバーフローで見比べて選ぶ。
   - **雅** … 明朝体の山名に英語名と標高を添える王道
   - **頂** … 山名を大きく据えるポスター風
   - **語** … 山の解説を日英併記で添える読み物風
   - **間** … 大きな余白に縦書きの解説、掛け軸のような作品風
   - **空** … 上に空色の余白を広げ、稜線へ溶かし込む縦構図
4. **仕上げる** … 山名ラベル・解説・タイトル・余白・切り抜きを微調整（ラベルや解説はドラッグで移動、解説本文は編集可）。
5. **保存する** … Canvas 2D で合成した JPEG を書き出し。スマホは共有シート、PCは直接ダウンロード。写真一覧から ZIP で一括保存もできる。

3Dは使わないため山頂位置の自動合わせはなく、ラベルは既定配置からドラッグで合わせる。

## 特徴

- **山岳辞書 27,371座** … 山名・読み・英名・標高を収録。読みはWeb検証で通用読みに修正済み
- **解説データベース 974座** … ヤマケイ・YAMAP・Wikipedia の事実情報をもとにAIが独自執筆（日英×長短の4種＋タグ、機械検品済み）。シャード分割で必要な分だけ配信
- **写真主体のUI** … 漆黒基調のフラットなデザイン。ホームは作例を敷き詰めた「動く写真の壁」、一覧・山選び・テーマ選択は画面端まで使う full-bleed レイアウト
- **編集状態の保存** … 各写真の編集状態はスナップショットとして保持され、一覧から何度でも再編集できる

## 開発

```bash
npm install
npm run dev       # 開発サーバ
npm run build     # 型チェック＋本番ビルド
npm run preview   # ビルド結果のプレビュー
npm run lint      # ESLint
```

React 19 + TypeScript + Vite。実行時依存は exifr（EXIF読み取り）のみ。

## 構成

- `src/App.tsx` … 画面ルーター（ホーム / 写真一覧 / 山選び / 仕上げ）と作業状態の管理
- `src/components/Picker.tsx` … ホーム（写真の壁ヒーロー・作例モザイク・写真選択）
- `src/components/MountainPicker.tsx` … 山選び（辞書検索・複数選択）
- `src/components/Board.tsx` … 写真一覧ハブ（状態表示・再編集・ZIP一括保存）
- `src/components/Studio.tsx` … 仕上げ（テーマ選択・編集・Canvas 焼き込み・書き出し）
- `src/lib/mountains.ts` … 山岳辞書の検索・解説シャードのロード
- `src/lib/labels.ts` … 選んだ山＋辞書解説 → 焼き込み用ラベルの組み立て
- `src/lib/exif.ts` … 写真の EXIF 読み取り
- `src/lib/zip.ts` … まとめて保存用の最小 ZIP ライタ（UTF-8名対応）
- `public/data/` … `mountains.json`（山岳辞書）/ `descriptions/`（解説シャード）
- `scripts/` … 辞書・解説データベースの構築パイプライン（取得・照合・検品・公開）

## 出典

山岳データは、あにねこ氏「山名一覧 on the Web地図」(map.jpn.org)・「日本の主な山岳標高一覧」（国土地理院）を加工。
解説文は、ヤマケイオンライン・YAMAP・Wikipedia の事実情報をもとにAIが独自に執筆したオリジナル（転載なし・機械検品済み）。
