# Frame — 山を写す

山の写真に山名ラベルや解説を重ね、印象的な一枚に仕上げるウェブアプリ。
[trace](../trace) の「山を写す(AR)」機能から、3Dマップ・撮影地点・向き合わせ（AR投影）を取り除き、
**「山名を選ぶ → 写真をのせる → テンプレートで仕上げる」** に絞った派生プロジェクト。

## 使い方（フロー）

1. **山名を選ぶ** … 辞書（約1,061山）から山名・読みで検索し、写真にのせる山を選ぶ（複数可）。
2. **写真をのせる** … 手元の写真を選ぶ。選んだ山の解説・英名・標高・タグが辞書から自動で付く。
3. **テンプレートを選ぶ** … 雅 / 頂 / 栞 / 双 / 間 / 空 の6種。文字・解説・余白をまとめて整える。
4. **仕上げる** … 山名ラベル・解説・タイトル・余白・切り抜きを微調整（ラベルや解説はドラッグで移動）。
5. **書き出す** … Canvas 2D で合成した JPEG をプレビューしてダウンロード。

3Dは一切使わないため、山頂位置は写真上に既定配置され、編集画面でドラッグして合わせる。

## 開発

```bash
npm install
npm run dev       # 開発サーバ
npm run build     # 型チェック＋本番ビルド
npm run preview   # ビルド結果のプレビュー
```

## 構成

- `src/App.tsx` … 画面ルーター（入口 ⇄ 仕上げ）
- `src/components/Picker.tsx` … 入口（山名検索・選択 → 写真選択）
- `src/components/Studio.tsx` … 仕上げ（テンプレ選択・編集・Canvas 焼き込み・書き出し）。trace の bakeComposite と可動編集を移植
- `src/lib/mountains.ts` … 山岳辞書の検索・解説ロード（trace から流用）
- `src/lib/labels.ts` … 選んだ山＋辞書解説 → 焼き込み用ラベルの組み立て
- `src/lib/exif.ts` … 写真の EXIF 読み取り（trace から流用）
- `public/data/` … `mountains.json`（山岳データ）/ `mountain_descriptions.json`（AI生成の解説）

## 出典

山岳データは「日本の主な山岳標高一覧」（国土地理院）を加工。解説文は事実情報をもとにAIで生成したオリジナル。
